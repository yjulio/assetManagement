❓ Frequently Asked Questions

Quick answers to common questions

How do I add a new asset to the system? ▼

To add a new asset:

  1. Click on Assets → Add an Asset in the sidebar
  2. Fill in the required fields (name, category, serial number, etc.)
  3. Upload images or documents if needed
  4. Click Submit to save

The asset will appear in your inventory immediately.

How do I check out an asset to an employee? ▼

Navigate to Assets → Check Out, select the asset, choose the employee, set the expected return date, and confirm. The system will track who has the asset and when it's due back.

Can I import multiple assets at once? ▼

Yes! Go to Tools → Import Data, download the CSV template, fill it with your asset data, and upload it. The system will import all assets in bulk.

How do I generate reports? ▼

Click on Reports in the sidebar and choose the type of report you need (Inventory, Depreciation, Maintenance, etc.). Set your filters and date range, then click Generate Report. You can export the report to PDF, Excel, or CSV.

What is asset depreciation and how is it calculated? ▼

Asset depreciation is the decrease in asset value over time. The system supports multiple methods:

  • Straight-line: Equal depreciation each year
  • Declining balance: Accelerated depreciation

You can set the depreciation method, useful life, and salvage value for each asset.

How do I schedule maintenance for an asset? ▼

Go to Assets → Maintenance, select the asset, enter maintenance details (type, cost, vendor), and set the next maintenance date. The system will track maintenance history and send reminders.

Can I track asset locations? ▼

Yes! Set up locations in Setup/Configuration → Locations, then assign assets to specific locations. You can also move assets between locations using Assets → Move.

How do I dispose of an asset? ▼

Navigate to Assets → Dispose, select the asset, choose disposal method (sold, donated, scrapped), enter details and reason, then confirm. The asset will be marked as disposed but remains in records for audit purposes.

What user roles and permissions are available? ▼

The system supports multiple user groups with different permissions:

  • Admin: Full access to all features
  • Manager: Can manage assets and generate reports
  • User: Can view assets and check in/out
  • Viewer: Read-only access

Configure user groups in Setup/Configuration → Groups.

How do I backup my data? ▼

Use Tools → Export Data to export all your data to CSV or Excel. For database backups, contact your system administrator. Regular backups are recommended to prevent data loss.

Can I customize the system for my organization? ▼

Yes! You can customize:

  • Company information and logo
  • Asset categories and subcategories
  • Custom fields on forms
  • Departments and locations
  • Email notifications

Access these options in Setup/Configuration.

What browsers are supported? ▼

The system works best on modern browsers:

  • Google Chrome (recommended)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

Make sure JavaScript is enabled for full functionality.

How do I reset my password? ▼

Contact your system administrator to reset your password. For security reasons, only administrators can reset user passwords.
